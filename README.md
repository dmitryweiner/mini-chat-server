# Simple chat demo server
Stores all its data to memory. Easy to erase.

## Technologies
* Express.js
* Jest
* Supertest

## How to use

* Install all dependencies.

  `npm install` 

* Run server.

  `npm run start` 

* Run tests.

  `npm run test` 

* Run ES linter.

  `npm run lint`
